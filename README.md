# Translator with educational functions - Переводчик с образовательными функциями

## Описание проекта

Translator with educational functions - это веб-приложение для перевода текста и изучения языка, разработанное для улучшения навыков написания текстов, анализа грамматики и расширения словарного запаса с использованием современных технологий и искусственного интеллекта.

## Особенности

- API для перевода текста и предоставления рекомендаций по улучшению
- Интерактивные викторины для изучения языка
- Интеграция с Google Gemini API для интеллектуального анализа
- Поддержка нескольких языков (включая русский и английский)
- Система транскрипции и текст-в-речь (TTS)

## Технологии

- **Backend**: Java, Spring Boot, Spring Data JPA
- **Frontend**: HTML, CSS, JavaScript
- **База данных**: PostgreSQL
- **AI/ML**: Google Gemini API, LanguageTool API, OpenNLP
- **Сборка**: Maven

## Быстрый старт

### Предварительные требования

- Java 17+
- Maven 3.6+
- PostgreSQL
- Node.js (для frontend разработки)

### Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/maks2134/maxify.git
```

2. Настройте переменные окружения в файле `server/src/main/resources/application.properties`

3. Установите зависимости:
```bash
cd server
mvn install
```

4. Запустите приложение:
```bash
mvn spring-boot:run
```

Приложение будет доступно по адресу `http://localhost:8080`

## Конфигурация

Приложение использует следующие переменные окружения:

### Backend
- `DB_URL` - URL базы данных PostgreSQL
- `DB_USERNAME` - Имя пользователя базы данных
- `DB_PASSWORD` - Пароль пользователя базы данных
- `GEMINI_API_KEY` - API ключ для Google Gemini
- `GEMINI_MODEL` - Модель Gemini для использования
