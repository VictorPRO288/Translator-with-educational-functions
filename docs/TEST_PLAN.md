## 6. Тестовые сценарии

### 6.1. Сценарий: Успешный перевод текста
**ID:** TC-001  
**Заголовок:** Успешный перевод текста с английского на русский  
**Предусловия:** 
- Пользователь зарегистрирован и аутентифицирован
- Внешнее API доступно

**Шаги:**
1. Пользователь вводит текст "Hello world" в поле ввода
2. Выбирает английский как исходный язык
3. Выбирает русский как целевой язык
4. Нажимает кнопку "Перевести"

**Ожидаемый результат:** Отображается перевод "Привет мир", и перевод сохраняется в истории

### 6.2. Сценарий: Создание викторины
**ID:** TC-002  
**Заголовок:** Создание викторины на основе перевода  
**Предусловия:** 
- У пользователя есть сохраненный перевод
- Пользователь аутентифицирован

**Шаги:**
1. Пользователь выбирает сохраненный перевод
2. Нажимает "Создать викторину"
3. Система генерирует вопросы
4. Пользователь проходит викторину

**Ожидаемый результат:** Викторина успешно создана и пройдена, результаты отображаются

### 6.3. Сценарий: Ошибка API
**ID:** TC-003  
**Заголовок:** Обработка ошибки при недоступности внешнего API  
**Предусловия:** 
- Внешнее API недоступно

**Шаги:**
1. Пользователь вводит текст для перевода
2. Нажимает кнопку "Перевести"

**Ожидаемый результат:** Отображается понятное сообщение об ошибке
