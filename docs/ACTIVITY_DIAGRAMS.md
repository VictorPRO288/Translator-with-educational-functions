# Диаграмма активности

## Основной процесс перевода

На диаграмме активности показан процесс перевода текста, начиная с ввода текста пользователем, перевода, создания квиза, его прохождения, грамматического анализа, получения транскрипции, озвучки перевода и заканчивая возможностью переводить дальше.
