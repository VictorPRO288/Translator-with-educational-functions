# Глоссарий терминов проекта "Translator with educational functions"

| Термин | Описание |
|--------|----------|
| Пользователь | Человек, который использует приложение для перевода текста |
| Перевод | Преобразование текста с одного языка на другой |
| Переводчик | Веб-приложение для перевода текста между различными языками |
| Исходный язык | Язык, с которого выполняется перевод текста |
| Целевой язык | Язык, на который выполняется перевод текста |
| Оригинальный текст | Текст на исходном языке, который нужно перевести |
| Переведённый текст | Текст на целевом языке, результат перевода |
| Квиз | Тест на знание перевода с 5 вопросами по переведённому тексту |
| Вопрос | Элемент квиза, проверяющий понимание конкретного аспекта перевода |
| Вариант ответа | Один из предложенных вариантов ответа на вопрос квиза |
| Правильный ответ | Верный вариант ответа на вопрос квиза |
| Оценка | Результат прохождения квиза в процентах |
| Подсказка | Грамматическая или стилистическая рекомендация по тексту |
| Анализ грамматики | Проверка текста на грамматические и орфографические ошибки |
| Анализ стиля | Проверка текста на стилистические ошибки |
| Транскрипция | Фонетическая запись текста с помощью международного фонетического алфавита (IPA) |
| Примеры использования | Примеры контекстного употребления слов и фраз |
